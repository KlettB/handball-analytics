# Phase 2: Einzelspiel-Tiefe

## Ziel
Tiefergehende Analyse einzelner Spiele: Runs, Momentum, Auszeit-Effektivität.

## Features (geplant)
- Run-Erkennung (z.B. "5:0 Lauf in 3 Minuten")
- Momentum-Visualisierung (Tordifferenz über Zeit)
- Auszeit-Effektivität (Ergebnis vor/nach Auszeit)
- Strafzeiten-Übersicht (wer saß wann)
- Halbzeit-Vergleich
